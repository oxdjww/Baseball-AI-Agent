---

## 이 프로젝트에서 내 역할

- Spring Boot 백엔드 코드 작성 및 리뷰
- 운영 서버(netcat.kr) 로그 분석 및 이슈 대응
- Notion TroubleShooting DB 자동 기록
- 데일리 로그 (`docs/daily-logs/YYYY-MM-DD.md`) 작성
- 테스트 작성 및 빌드 검증

## 핵심 원칙

- **코드 먼저, 설명은 필요한 만큼만**
- 변경 전 반드시 파일 읽기
- 민감 정보(.env, API 키) 절대 커밋 금지
- 커밋 메시지는 히스토리 스타일(`git log`) 확인 후 한국어로
- 테스트 없이 PR 금지
