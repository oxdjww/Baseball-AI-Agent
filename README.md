## 🚀 프로젝트 소개
Baseball AI Agent는 **KBO 실시간 경기 이벤트**를 수집하고,  
**텔레그램으로 즉시 알림**을 전달하는 서비스입니다.  
또한 경기 종료 후에는 **AI 기반 경기 분석 리포트**까지 자동 전송합니다.

- ✅ 경기 3시간/1시간 전 우천 취소 자동 감지 및 공지  
- ✅ 동점/역전 등 주요 이벤트 실시간 알림  
- ✅ 경기 종료 후 1시간 뒤, AI 분석 레포트 제공
